课程：基本 I/O

本课程介绍用于基本 I/O 的 Java 平台类。它首先关注 I/O Streams (I/O 流)，这是一个强大的概念，可以大大简化 I/O 操作。本课程还介绍了序列化，它允许程序将整个对象写入流并再次读回。然后，本课将介绍文件 I/O 和文件系统操作，包括随机访问文件。

I/O Streams 部分中涵盖的大多数类都在 java.io 包中。File I/O 部分中涵盖的大多数类都在 java.nio.file 包中。

I/O 流

  • 字节流处理原始二进制数据的 I/O 。
  • 字符流处理字符数据的 I/O，自动处理与本地字符集的转换。
  • 缓冲流通过减少对原生 API 的调用次数来优化输入和输出。
  • 扫描和格式化允许程序读取和写入格式化文本。
  • 来自命令行的 I/O描述标准流和控制台对象。
  • 数据流处理基本数据类型和 String 值的二进制 I/O。
  • 对象流处理对象的二进制 I/O 。

文件 I/O(采用 NIO.2)

  • 什么是路径？检查文件系统上路径的概念。
  • 路径类介绍 java.nio.file 包的基石类。
  • 路径操作查看 Path 类中处理语法操作的方法。
  • 文件操作引入了许多文件 I/O 方法的共同概念。
  • 检查文件或目录显示如何检查文件的存在及其可访问性级别。
  • 删除文件或目录.
  • 复制文件或目录.
  • 移动文件或目录.
  • 管理元数据说明了如何读取和设置文件属性。
  • 读取、写入和创建文件显示用于读取和写入文件的流和通道方法。
  • 随机访问文件显示了如何以非顺序方式读取或写入文件。
  • 创建和读取目录涵盖了特定于目录的 API，例如如何列出目录的内容。
  • 链接，符号或其他涵盖特定于符号和硬链接的问题。
  • 遍历文件树演示了如何递归访问文件树中的每个文件和目录。
  • 查找文件显示如何使用模式匹配搜索文件。
  • 观察目录中的更改显示了如何使用监听服务来检测在一个或多个目录中添加，移除或更新的文件。
  • 其他有用的方法涵盖了本课程其他部分不适用的重要 API。
  • 旧版文件 I/O 代码显示如果使用 java.io.File 类的旧代码，如何利用 Path 功能。提供了将 java.io.File API 映射到 java.nio.file API 的表。
